### READ ME ### 

## Requirements ##
1. Java 17
2. node.js and npm 
3. VS code

## Running the Code ##
1. Check out the read me inside of frontend for more!
2. Open a new terminal window (Ctrl + `) 
3. cd frontend
4. npm install
5. npm run serve 
6. To stop the server, Ctrl + C.
